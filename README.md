#CLI Reminders App:
The user should be able to input a reminder, the app should be able to store the reminder until completed, and app should be self-contained.
##The CLI Reminders Project is an app that allows users to be able to input multiple reminders, as well as store the dates the reminders are due. The project should be able to keep track of dates, and should be self-contained. This project exists because helps people remember their tasks by inputing thier reminders within the app. Tools that were used to create this app was Visual Studio Code.
##The code can be run by clicking the "Run code" button, then you will see a Welcome screen that says "CLI Reminders", and an ASCII art. The color for these two images are blue, but the color of the text of the program is magenta. The code will ask you to input a number from 1-4(1 to set reminders, 2 to view reminders, 3 to delete selected reminders, and 4 to exit the code). If you were to set a reminder, you will input 1, and you will see an input from the computer requesting you to enter a reminder. It will then ask you  to set a due date, or when the reminder is schedule to be completed, and then it should say "Reminder added successfully!", and it will take you back to the options menu.

##Some features of the app is the ASCII image of a car, the blue-colored title and image, as well as the magenta-colored text of the code's output. The code also calculates leap years, and states the time that you access the menu.
